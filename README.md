## Installs

### Npm modules
```npm install```

### Angular
```npm install -g @angular/cli```

### Charts.js
```npm install ng2-charts chart.js --save```

### Taillwind
```npm install -D tailwindcss postcss autoprefixer```

### Firebase 
```npm install -g firebase-tools```

### body parser
```npm install body-parser```
